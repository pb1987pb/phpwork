

   注释：
 

  1，HTML 注释：  <!--  html注释  --!>

  2, css 注释 ：  /*  css注释 */

  3,Javascript注释：

         a) 单行注释 :  // 单行注释

         b)  多行注释 ： /* 多行注释 */

    4,PHP 注释


       a) 单行注释 :  // 单行注释    #单行注释

         b)  多行注释 ： /* 多行注释 */
   

   5，mysql 注释：

   a) 单行注释：  -- 单行注释

  b) 多行注释 ： /*    多行注释  */
  
  经典面试题：
问：ThinkPHP中的行注释和块注释与普通的html注释有什么区别？

答：普通的html注释属于客户端注释，会在浏览器的源代码中输出；

而ThinkPHP中的模版注释则属于服务器端的注释，不会被浏览器输出。
