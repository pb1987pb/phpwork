
  数据库在插入的时候，要么设置默认值，要么系统会自动设置默认值，因为插入必须有数据。

   1，如果设置了 null ,也就是 默认值是 null 

   2, 如果设置 not  null ， 而且 也没有设置默认值 ，那么 系统就会自动 设置 默认值 插入数据库 。
   
      这个 默认值的 规则 是 

        int  -> 0 

       decimal(10,2)-> 0.00

        char ->''      空字符串

        datetime -> 0000-00-00 00:00:00

        enum('男','女')      默认第一个  男 

         date ->0000-00-00 

        set('足球，排球') ->''    空字符串 


   3，text 我们不能设置默认值  ，但是 在实践中 发现 ，如果 null ，那么默认就是 null ,如果 not null ，

          那么默认值就是 '' 空字符串
